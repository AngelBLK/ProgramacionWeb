¿Qué son las bases de datos?

"Las bases de datos se pueden definir como una colección de datos o información estructurada almacenados comúnmente en un sistema electrónico. Las bases de datos, al tener gran cantidad de información ordenada, nos facilitan el uso y el cómo encontrar datos de forma rápida y sencilla"(maldeadora, 2016).
